Ajuda Problemas com o pagamento? Caso você tenha algum problema com o pagamento, ou você ira adicionar uma nova forma de pagamento ou tentar solucionar o problema com as seguintes dicas : Verifique se os números inseridos no cartão de crédito estão corretos, data de validade, nome, e código de segurança. Se o cartão estiver expirado, é necessário deletar e adicionar um novo. Verifique se existe saldo em sua conta para efetuar o pagamento. Para eventuais problemas e demais dúvidas entre em contato conosco. Problema com a localização? Caso não esteja sendo possível conectar ao GPS, verifique se o modo 'alta precisão' está ativo Verifique se o celular possui conexão, ou via rede móvel ou via wi-fi. Verifique se o GPS está ativo Para outros problemas por favor entre em contato.
